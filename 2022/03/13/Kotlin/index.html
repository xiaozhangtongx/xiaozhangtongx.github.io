1. Kotlin 入门

1.1 Kotlin简介

Kotlin 是一种在 Java 虚拟机上运行的静态类型编程语言，被称之为 Android 世界的Swift，由 JetBrains 设计开发并开源。

Kotlin 可以编译成Java字节码，也可以编译成 JavaScript，方便在没有 JVM 的设备上运行。在Google I/O 2017中，Google 宣布 Kotlin 成为 Android 官方开发语言。

1.2 Kotlin 基本数据类型

Kotlin 的基本数值类型包括 Byte、Short、Int、Long、Float、Double 等。不同于 Java 的是，字符不属于数值类型，是一个独立的数据类型。

每种数据类型都有下面的这些方法，可以转化为其它的类型：

toByte(): Byte
toShort(): Short
toInt(): Int
toLong(): Long
toFloat(): Float
toDouble(): Double
toChar(): Char

    // 1.整型

1.3 Kotlin数据容器

1.3.1 数组

  • 数组在定义的时候要指定长度

  • 数组用类 Array 实现，并且还有一个 size 属性及 get 和 set 方法，由于使用 [] 重载了 get 和 set 方法，所以我们可以通过下标很方便的获取或者设置数组对应位置的值。

  • 数组的创建两种方式：一种是使用函数arrayOf()；另外一种是使用工厂函数。如下所示，我们分别是两种方式创建了两个数组

        // 1.使用arrayOf创建数组，必须要指定数组元素
    val a = arrayOf(1, 2, 3)
    val arrayNumber: Array<String> = arrayOf("i", "love", "you")
    // Any等价于Object
    val arrayObjects: Array<Any> = arrayOf(1, "45", '2', true)
    println(arrayObjects[1]) // 45
    // 2.听过arrayOfNulls来创建数组
    val arrayOfNulls = arrayOfNulls<String>(5)
    arrayOfNulls[0] = "xiaozhang"
    println(arrayOfNulls[0]) // xiaozhang
    // 3.动态创建数组，一般不用
    val b = Array(3) { i -> (i * 2) }
    println(b[1]) // 2

  • 数组的遍历

        // 数组的遍历
    // 1.用for ( in )的方式
    for (i in arrayObjects) {
        println(i)
    }
    // 2.下标的遍历
    for (i in arrayObjects.indices) {
        println(arrayObjects[i])
    }
    // 3.同时遍历下表和元素
    for ((index, i) in arrayObjects.withIndex()) {
        println("${index}:${i}")
    }
    // 4.forEach来遍历
    arrayObjects.forEach { item -> println(item) }
    // 5. forEach增强版
    arrayObjects.forEachIndexed { index, item -> println("$index->$item") }
    println("item")

1.5 条件控制

  • if-else
fun main() {
    val max = maxOf(1, 3)
    val max2 = maxOf2(1, 3)
    println(max)
    println(max2)

}

// 取两个数的最大值
fun maxOf(a: Int, b: Int): Int {
    if (a > b) {
        return a
    } else {
        return b
    }
}

// maxOf方法的改写，类似于三元表达式
fun maxOf2(a: Int, b: Int): Int {
    return if (a > b) a else b
}
  • shen

Kotlin中的when表达式可以认为是Java switch 语句的代替。它评估许多代替方案中的一段代码。

fun main(args: Array<String>) {

    val a = 12
    val b = 5

    println("输入运算符 +, -, * 或 /")
    val operator = readLine()

    val result = when (operator) {
        "+" -> a + b
        "-" -> a - b
        "*" -> a * b
        "/" -> a / b
        else -> "$operator 运算符是无效的运算符。"
    }

    println("result = $result")
}

1.6 泛型

泛型，即 “参数化类型”，将类型参数化，可以用在类，接口，方法上。

与 Java 一样，Kotlin 也提供泛型，为类型安全提供保证，消除类型强转的烦

1.6.1 泛型接口/类
fun main(args: Array<String>) {
    // 泛型接口
    DrinkApple().drink("apple")
    // 泛型类
    BlueColor("blue").printColor()
}


// 泛型接口
interface Drink<T> {
    fun drink(t: T)
}

class DrinkApple : Drink<String> {
    override fun drink(t: String) {
        println("drink:${t}")
    }
}

// 泛型类
abstract class Color<T>(val t: T/*泛型字段*/) {
    abstract fun printColor()
}

class BlueColor(val color: String) : Color<String>(color) {
    override fun printColor() {
        println(color)
    }
}
1.6.2 泛型方法
fun main(args: Array<String>) {
    val age = 23
    val name = "nhooo"
    val bool = true
    doPrintln(age)    // 整型
    doPrintln(name)   // 字符串
    doPrintln(bool)   // 布尔型
}
fun <T> doPrintln(content: T) {
    when (content) {
        is Int -> println("整型数字为 $content")
        is String -> println("字符串转换为大写：${content.toUpperCase()}")
        else -> println("T 不是整型，也不是字符串")
    }
}

1.7 Kotlin 扩展函数

假设您需要扩展具有新功能的类。在大多数编程语言中，您要么派生一个新类，要么使用某种设计模式来执行此操作。

但是，在Koltin中，您也可以使用扩展函数来扩展具有新功能的类。基本上，扩展函数是在类外部定义的类的成员函数。

fun String.removeFirstLastChar(): String =  this.substring(1, this.length - 1)

fun main(args: Array<String>) {
    val myString= "Hello Everyone"
    val result = myString.removeFirstLastChar()
    println("输出结果: $result")
}
