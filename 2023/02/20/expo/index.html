【RN】Expo使用react-navigation进行页面跳转及踩坑

xiaozhangtx 2/20/2023 RN

由于开发需求，最近在学react native，今天在Expo使用react-navigation时遇到了一些坑，这里记录一下

1 安装React Navigation

1.1 安装依赖包

yarn add @react-navigation/native
1

1.2 安装Expo的一些依赖

expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
1

从 React Native 0.60 及更高版本开始,不需要运行react-native link。其它的具体文档可以参考官方文档 (opens new window)

2 测试及使用

2.1 编写测试页面

这里为了简单直接使用官方文档给的例子

import * as React from 'react';
import { Button, View, Text } from 'react-native';
import { NavigationContainer } from '@react-navigation/native';
import { createNativeStackNavigator } from '@react-navigation/native-stack';

function HomeScreen({ navigation }) {
  return (
    <View style={{ flex: 1, alignItems: 'center', justifyContent: 'center' }}>
      <Text>首页</Text>
      <Button
        title="详情页面"
        onPress={() => navigation.navigate('Details')}
      />
    </View>
  );
}

function DetailsScreen() {
  return (
    <View style={{ flex: 1, alignItems: 'center', justifyContent: 'center' }}>
      <Text>Details Screen</Text>
    </View>
  );
}

const Stack = createNativeStackNavigator();

function App() {
  return (
    <NavigationContainer>
      <Stack.Navigator initialRouteName="Home">
        <Stack.Screen name="Home" component={HomeScreen} />
        <Stack.Screen name="Details" component={DetailsScreen} />
      </Stack.Navigator>
    </NavigationContainer>
  );
}

export default App;

3 遇到的坑

3.1 安卓无法打开

配置没有报错，但是一直无法加载出页面

解决方法：重启电脑，清理一下缓存，可以试试

3.2 报错

我按照官网的配置，但是就给一直报Unable to resolve "@react-navigation/native&quot; from "App.js" 这个错误，我找了一些解决方案，也尝试过重新安装依赖，但是都没有解决这个报错；就当我快睡觉的时候，突然之间之前报错的程序就可以运行了，真有点莫名其妙！这里给一个我参考过的解决方案 (opens new window)，可以看看，说不定能解决你的问题。

写在最后，有些时候代码不能运行，有可能不是代码的问题，试试重启，重启能够解决90%的问题！！！
